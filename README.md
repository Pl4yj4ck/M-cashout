# 🎲 M-cashout

![GitHub stars](https://img.shields.io/github/stars/Pl4yj4ck/M-cashout?style=for-the-badge)
![GitHub forks](https://img.shields.io/github/forks/Pl4yj4ck/M-cashout?style=for-the-badge)
![License](https://img.shields.io/github/license/Pl4yj4ck/M-cashout?style=for-the-badge)

Un semplice **tracker per gestire i soldi durante le partite di Monopoly** in modo digitale, veloce e intuitivo.

Perfetto per evitare errori nei conti e rendere le partite più fluide 🚀

---

## 📌 Funzionalità

- 💸 **Pagare un giocatore**
- 💰 **Ricevere pagamenti**
- 📊 **Visualizzare il saldo attuale**
- 🧾 **Cronologia delle partite**

---

## 🛠️ Stato del progetto

La funzione di **cronologia delle partite** è ancora in sviluppo.
